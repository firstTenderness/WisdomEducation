# UI/UX 设计技能

## 技能信息
- **名称**: ui-ux-design-skill
- **版本**: 1.0.0
- **作者**: Trae AI
- **描述**: 提供 UI/UX 设计相关的专业知识和工具，帮助优化项目的美观度和用户体验
- **标签**: design, ui, ux, frontend

## 功能介绍
- 提供 UI/UX 设计最佳实践
- 优化响应式设计
- 提升视觉效果和用户体验
- 提供色彩方案和设计系统建议
- 优化移动端和桌面端显示
- 自动调用技能：每次用户提出需求时，系统会自动调用相关技能来完善需求，无需用户明确请求调用技能

## 使用方法
1. 询问关于 UI/UX 设计的问题
2. 请求优化特定页面的设计
3. 寻求色彩方案和排版建议
4. 咨询响应式设计的最佳实践
5. 自动调用：系统会根据用户的需求自动调用相关技能，无需用户明确请求

## 示例

### 示例 1: 优化移动端响应式设计
**用户**: 如何优化 PortalHome.vue 的移动端显示？
**技能**: 可以通过以下步骤优化：
1. 使用媒体查询适配不同屏幕尺寸
2. 实现移动端菜单功能
3. 优化按钮和交互元素的大小
4. 调整布局和间距

### 示例 2: 提升视觉效果
**用户**: 如何提升课程卡片的视觉效果？
**技能**: 可以通过以下方式提升：
1. 添加渐变背景
2. 实现卡片悬停效果
3. 添加适当的阴影和过渡动画
4. 优化色彩对比和排版

## 技术建议
- 使用 CSS 变量管理颜色方案
- 采用 Flexbox 和 Grid 布局
- 实现平滑的过渡动画
- 确保触摸目标足够大（至少 44x44px）
- 优化图片加载和显示

## 设计原则
- **一致性**: 保持设计元素的一致性
- **可用性**: 确保界面易于使用
- **可访问性**: 确保所有用户都能访问内容
- **性能**: 确保设计不会影响性能
- **美观**: 创建视觉吸引力强的界面
